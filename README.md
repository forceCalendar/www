# forceCalendar Website

Marketing and documentation website for the forceCalendar enterprise calendar infrastructure.

## Overview

This website showcases the forceCalendar project:
- **@forcecalendar/core** - Pure JavaScript calendar engine
- **@forcecalendar/interface** - Web Components UI layer

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
